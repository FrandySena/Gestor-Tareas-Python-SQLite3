# Gestor de tareas - Python y SQLite

Este es un proyecto de consola desarrollado en Python que permite gestionar tareas pendientes, permitiendo agregar, buscar, actualizar y eliminar registros utilizando una base de datos SQLite.

## Caracteristicas
- **Validacion de fechas al usuario**: Formato estricto *YYYY-MM-DD* y validación para que la fecha final no sea anterior a la actual.
- **Base de datos:** Uso de SQLite para persistencia de datos.
- **Prioridades y Estados:** Clasificación por prioridad (Alta, Media, Baja) y estados (Pendiente, En Progreso, Completada).

## Como ejecutar
1. Asegúrate de tener los archivos *main.py* y *GestorTareas.sql* en el proyecto.
2. Ejecuta el archivo *main.py*.
3. Se abrira el menu en la consola.

## Tecnologias
- Python 3
- SQLite3
- Datetime (Manejo de fechas)
